# ShowMyData

ShowMyData is a collection of free, open-source web applications for creating, exploring, and teaching data-showing graphs. The apps are designed to make it easy to visualize individual data points, understand statistical relationships, and communicate data clearly in educational, research, and applied settings.

## About the Project

ShowMyData was created by Jeremy B. Wilmer to promote the use of data-showing graphs in science, education, and data analysis. Each application is designed to be easy to use while providing publication-quality visualizations that emphasize the underlying data rather than summary statistics alone.

The applications are freely available through **ShowMyData.org**, and the complete source code for each app is available in this GitHub organization.

Contributions, suggestions, bug reports, and feature requests are welcome.
